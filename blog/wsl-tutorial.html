Cara menjalankan program C++ di Windows Subsystem Linux (WSL)

Pengertian WSL

Menurut wikipedia, Windows Subsystem for Linux merupakan lapisan kompatibilitas untuk menjalankan biner linux yang dapat dijalankan (dalam bentuk ELF) secara alami di Windows 10.

Cara Install WSL

  1. Install WSL
    • Buka Windows Powershell, run as administrator
    • Tambahkan command ini di Windows Powershell
    • Tunggu sampai instalasi selesai dan restart komputer Anda
  2. Install Distro Linux (Ubuntu)
    • Buka Microsoft Store
    • Cari dan download Ubuntu 20.04 LTS
    • Buka Ubuntu 20.04 LTS kemudian buat UNIX username dan password
  3. Install C++ compiler menggunakan g++ cek apakah g++ sudah terinstall atau belum menggunakan script ini
  4. Install visual studio code (VS Code) disini
  5. Install extension remote WSL di VS Code
  6. Buka WSL, jalankan script ini
